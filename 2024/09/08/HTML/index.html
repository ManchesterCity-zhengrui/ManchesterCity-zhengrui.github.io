HTML

HTML超文本标记语言–HyperText Markup Language。

超文本？mean：链接

标记？也叫标签，带尖括号的文本
标签的语法

       javascript //指定语言 int main() { const int n = 10; printf("hello csdn!"); return 0; }

标签

加粗

加粗 语法： 需要加粗的文字 可以不用strong，只要前后一致即可。 (绝大多数)双标签：成对出现的标签 单标签：
:换行
:水平线

HTML基本骨架

HTML基本骨架是网页模板

网页标题 网页主体 以上的 html：整个网页 head：网页头部，存放给浏览器看的代码，例如css body：网页主体，存放给用户看的代码，例如图片文字 title：网页标题

快速生成基本骨架：！（英文）+回车（或tab）

兄弟关系（并列关系）

注释

ctrl + /

标题标签

标题标签一般用于新闻标题，文章标题，网页区域名称，产品名称等
标签名：h1h6（双标签）六级六个标题，独占一行
h1标签在一个网页中只能用一次，用来放新闻标题或网页的logo
h2h6没有使用次数限制

段落标一般用在新闻段落、文章段落、产品信息描述等
