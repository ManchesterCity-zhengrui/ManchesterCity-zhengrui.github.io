Markdown

Author: Zheng Rui Date: July 20, 2024  8:02:47

Markdown 是一种轻量级标记语言，具有简洁、易读易写的特点，常用的标记符号不超过十个。以下是一些常见的 Markdown 语法：

  1. 标题：使用 # 号标记标题，# 的个数表示标题级别，一个 # 对应一级标题，两个 # 对应二级标题，以此类推。例如：

  1. 链接：有行内式和参考式两种形式。
    • 行内式链接：在方括号 [] 内写链接文字，紧接着在圆括号 () 内写链接地址，如果需要添加链接标题，可在链接地址后用双引号括起来，例如：

  • 参考式链接：在链接文字的方括号 [] 后再接上另一个方括号 []，在第二个方括号里面填入用以辨识链接的标记（可自定义）。然后在文件的任意处，通过方括号（前面可选择性地加上至多三个空格来缩进）、里面输入链接文字、接着一个冒号、一个以上的空格或制表符、链接的网址来定义链接内容。如果需要添加链接标题，可以用单引号、双引号或是括弧包着放在网址后面。此外，还有隐式链接标记，只需在链接文字后面加上一个空的方括号 [] 即可。

  1. 上标和下标：上标使用 ^ ，下标使用 ~ ，例如：x^2 表示上标2，H~2~O 表示下标2。
  2. 数学公式：使用 $$ 包裹公式内容，例如：$$E=mc^2$$ 表示质能方程。对于行内公式，可以使用 $ 包裹，如 $e=mc^2$ 。
  3. 转义字符：如果要显示特殊符号本身，可在其前加上反斜杠 \ 进行转义。

Markdown 还支持一些高级用法，如绘制流程图、序列图、甘特图等，具体语法可以根据使用的 Markdown 编辑器或相关插件进行学习和使用。不同的 Markdown 编辑器可能会对语法有一些细微的差异或扩展，但基本语法是相似的。

如果你想更详细地了解 Markdown 语法，可以参考相关的 Markdown 教程或编辑器的帮助文档。同时，通过实际使用和练习，你可以更快地掌握 Markdown 的语法并灵活运用它来编辑文档。
